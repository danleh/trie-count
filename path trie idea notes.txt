# pt (short for pathtrie)

- input: lines from stdin or file (if given)
- ouput: tree-like output, path components indented by depth, e.g.
/
    usr/
        bin/
            bash
            sh
- optional output: counts of direct/transitive children
- implemented in Rust, with clap v3
- path tree datastructure, where each node contains a path component, i.e. struct { component: String, children: Vec<Node> }, where children are ordered by their contains prefix
